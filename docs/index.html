Abstract

Topic modeling seeks to uncover latent semantic structure in text, with LDA providing a foundational probabilistic framework. While recent methods often incorporate external knowledge (e.g., pre-trained embeddings), such reliance limits applicability in emerging or underexplored domains. We introduce PRISM, a corpus-intrinsic method that derives a Dirichlet parameter from word co-occurrence statistics to initialize LDA without altering its generative process. Experiments on text and single cell RNA-seq data show that PRISM improves topic coherence and interpretability, rivaling models that rely on external knowledge. These results underscore the value of corpus-driven initialization for topic modeling in resource-constrained settings.

Paper: OpenReview PDF

Method

PRISM is a corpus-intrinsic initialization method for LDA. It constructs a PPMI-based word similarity graph from document-level co-occurrence, extracts diffusion-map embeddings (anisotropic normalization), softly clusters embeddings with a GMM to obtain topic-word distributions, estimates a Dirichlet prior β̂ via method-of-moments, and initializes LDA with β̂—providing prior-like guidance while keeping LDA's generative process unchanged.

WID: Word Intrusion Detection

Word Intrusion Detection (WID) evaluates topic interpretability by asking an evaluator to identify an out-of-place word among a topic’s top words (the “intruder”). To scale this human-centric task, we use an LLM as an automated judge (following prior work), prompting it to select the intruder for each topic instance.

Installation

1) Clone this repo

git clone https://github.com/shaham-lab/PRISM.git
cd PRISM

2) Run the setup script (downloads + patches + builds MALLET)

chmod +x setup_mallet.sh
./setup_mallet.sh

This downloads MALLET 2.0.8 source, renames it to ./mallet, replaces ./mallet/src/cc/mallet/ with the patched directory (beta_vector_patch/mallet), then runs ant clean && ant.

3) Verify it worked

cd mallet
./bin/mallet --help

Troubleshooting (common)

  • ant: command not found → install Apache Ant, then rerun the script.
  • Java issues → check java -version and javac -version, ensure a working JDK is installed.

Quickstart

Your entry point is src/main.py.

Run from repo root:

python src/main.py <DATASET> <METRIC> <NUM_TOPICS> <NUM_ITERATIONS> <RUN_NUMBER> <SEED>

Example:

python src/main.py BBC prism 5 1000 1 42

Alternatively: cd src then python main.py ...
